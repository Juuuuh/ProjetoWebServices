# Projeto Web Services — Spring Boot, JPA e Hibernate

Este projeto foi desenvolvido com o objetivo de aprender a criar **Web Services em Java** utilizando **Spring Boot**, **JPA** e **Hibernate**.

Ao longo do desenvolvimento são aplicados conceitos importantes como a criação de APIs REST, organização da aplicação em camadas e persistência de dados com ORM.

---

## Objetivos do Projeto

- Criar um projeto com Spring Boot
- Implementar o modelo de domínio
- Estruturar a aplicação em camadas (resource, service, repository)
- Configurar base de dados de teste com H2
- Utilizar JPA e Hibernate para persistência de dados
- Implementar operações CRUD
- Implementar tratamento de exceções

## Estrutura do Projeto

O projeto está organizado por camadas:

- resource
- service
- repository
- entities
- config

### Descrição das Camadas

**Resource**  
Responsável pelos endpoints da API (controladores REST).

**Service**  
Contém a lógica de negócio da aplicação.

**Repository**  
Responsável pelo acesso à base de dados através do JPA.

**Entities**  
Classes que representam as entidades do sistema e as tabelas da base de dados.

**Config**  
Configurações da aplicação e inicialização da base de dados.
